Upload profile pictures for each team member in their respective folders.

## Image Specifications

| Size | Dimensions | Use Case |
|------|------------|----------|
| small.jpg | 400 × 500 px | Mobile devices |
| medium.jpg | 800 × 1000 px | Desktop (recommended) |
| large.jpg | 1200 × 1500 px | High-res/Retina displays |

## Requirements
- **Aspect Ratio**: 4:5 (portrait)
- **Format**: JPG or WebP
- **File Size**: < 200 KB per image (optimized)
- **Style**: Professional headshot, head and shoulders
- **Background**: Neutral preferred (grayscale filter will be applied)

## Naming Convention
- Use lowercase filenames
- Use `small.jpg`, `medium.jpg`, `large.jpg` for consistency
- Alternative: `profile-400.jpg`, `profile-800.jpg`, `profile-1200.jpg`
